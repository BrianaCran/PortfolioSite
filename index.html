Prey of the Gods

Prey for the Gods Project

Summary

The Prey of the Gods is a game currently being developed by a gaming studio comprised of only three developers. They have gotten past the first hurrdle of acquiring funding and put together a quick website. This website is good enough to convey information to those looking for it, but because of time constraints faced by the developers it lacks in the quality that most gamers are used to seeing and the organization of information could be presented in a better way.

Challenges

The challenges that came along with this project was to bring it up to the quality of other gaming sites while also staying within branding in order to not alienate current backs. Since the game is still being developed the assets available for use are limited.

How to make it successful?

Using existing colors to stay within branding. Creating complimentary assets to bring more contrast to the current layout. Bringing more attention to current assets by adding a difference between them and website background. Spicing up the website by spreading existing assets to other pages. Putting everything together to create an interesting and user friendly site.

Scoundrels of the Atlantic

Scoundrels of the Atlantic

Summary

The Museum of Natural Sciences in North Carolina is wishing to increase interest in state history and exploring the museum. They requested a separate responsive website based off an exhibit, current or previous, that would be interactive and lead back to the main museum website.

Challenges

This came with quite a few challenges. First is to design a responsive interactive site that would reach out to younger demographics. Second is to generate interest in learning more and possible nudging that curiosity towards the museum.

How to make it successful?

Creating a few important set pieces that requires user interactive to engage curiosity. Have a prominent link back page to the museum. Provide responsive capabilities to reach both desktop and mobile users.

Goodreads

Goodreads

Summary

Goodreads is an online website that promotes interacting with authors and other book readers. One of the biggest features of the site is the abillity to add books to lists of currently reading, have read, and to read. These options make up the bookself page which was in need of a redesign.

Challenges

The challenge of this redesign was not only to reorganize all the information, but to keep within the existing site branding. It needed to be fresh and visually pleasing to those new to the site and current user, but at the same time current user need to be taken into consideration so that they do not feel alienated by the site redesign.

Success

I stuck to the colors used by the site to keep within branding standards. I provided easy readability by breaking the body information into three different columns. The link styles were keep the same in order to provide familiarity for current users. The currently reading section images are attached to a modal that ideally given enough time would be added for the rest of the book images so that users can't get more details and options without leaving the bookshelf page.

Challenges

This came with quite a few challenges. First is to design a responsive interactive site that would reach out to younger demographics. Second is to generate interest in learning more and possible nudging that curiosity towards the museum.

How to make it successful?

Keep the design simple and include enough critical actions to keep things less confusing for wider age audiences. But, keep customers with additional options. Adding options like 24-7 customer service and user pictures when transferring between branchless users so no test payment needs to be sent.
